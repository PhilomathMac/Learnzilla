## Description
A swiftUI app that teaches the basics of both Swift and SwiftUI using apple's documentation, videos, and quizzes.

## Skills Practiced

* EnvironmentObjects
* LazyVStack
* .onAppear Modifier
* AVPlayer
* AVKit
* .clipShape Modifier
* Authentication Flow
* Error Message Flow
* Firebase Authentication
* Firebase Database
* Observable Objects
* NSAttributedStrings
* UserService Meta Data
* JSON

## Future Features / Fixes

## Use Instructions

This app involves using a Firebase database. 
1. Clone project
2. Import Firebase dependencies
3. Create a Firebase project and download GoogleServices-Info.plist
4. Add plist to project
5. Ensure Firebase project has email/password authentication turned on
